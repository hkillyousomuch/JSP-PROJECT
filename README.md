 ## 로그인
    - 회원가입
     > 아이디, 비밀번호, 이름, 생년월일, 이메일(google, naver 만), 주소, 전화번호
    - 로그인
     > 아이디, 비밀번호, 아이디 및 비밀번호 찾기
    - 회원정보
 ### 아이디 및 비밀번호 찾기
   - 아이디 찾기
    > 이름, 이메일 입력
    --> 이메일로 아이디의 앞, 뒤 3자리 씩 출력 이외의 정보 (*) 표시
   - 비밀번호 찾기
    > 이름, 아이디, 이메일 입력
    --> 이메일로 비밀번호 변경 페이지 전송(google, naver 정도로 제한)
 ## 회원정보
    - 비밀번호 변경 (원래 사용하던 비밀번호 입력 시 변경 가능)
    - 주소 변경(배송지)
    - 전화번호 변경
